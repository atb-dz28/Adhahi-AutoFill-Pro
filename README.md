# 🕌 Adhahi AutoFill Pro

[![Chrome Web Store](https://img.shields.io/badge/Chrome-Extension-blue.svg)]()
[![License](https://img.shields.io/badge/License-MIT-green.svg)]()

Chrome extension for automatic reservation form filling on Adhahi.dz platform.

## ✨ Features

- 🔄 Automatic form filling (NIN, ID, Phone, Email, Password)
- 👥 Multi-client management with profile switching
- 📊 Real-time availability monitoring (updates every 1 second)
- 🔔 Smart notifications when new wilayas become available
- ✅ Real-time validation with visual feedback
- 🏙️ Complete database of 58 wilayas and their communes
- 🎮 Simulation mode for testing

## 🚀 Quick Start

1. Clone or download this repository
2. Go to `chrome://extensions/` in Chrome
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension folder
